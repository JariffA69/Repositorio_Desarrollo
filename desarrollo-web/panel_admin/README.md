Se realizó la nueva ventana de panel admin para que al momento de loguearse un administrador lo dirija a esta ventana y no a la home page que es para los otros usuarios en general (mecanicos/clientes).

Dentro de este panel admin se encontrará desplegada la lista de usuarios registrados en la base de datos y tendrán sus botones/opciones para editar y eliminar a cualquier usuario que se desee.

De igual forma en la parte debajo de la lista de usuarios se encuentra un pequeño formato de formulario sencillo el cual cumple la función de ser rellenado para luego dar click al boton de "crear usuario" y pueda ser añadido tanto a la lista de panel admin como a la base de datos mysql.

*Se anexan files del panel admin (html, css, ts)
