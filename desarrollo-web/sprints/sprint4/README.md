# Desarrollo gestion de usuarios

## Objetivo del sprint
Organizar y documentar el progreso del proyecto, actualizar la estructura de la base de datos y establecer los avances necesarios para continuar con el desarrollo en los siguientes sprints.

## Descripcion
Este repositorio contiene las evidencias correspondientes al Sprint 4 del Proyecto Integrador, desarrolladas siguiendo la metodología ágil SCRUM. Durante este sprint se trabajó en la organización del proyecto, la planificación de tareas y la actualización de artefactos técnicos necesarios para continuar con el desarrollo.
Las evidencias se encuentran en /evidencias/ donde encontraran el documento con nuestros avances.
