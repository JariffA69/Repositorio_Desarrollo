# Disenio de wireframes y prototipo de figma

## Objetivo
Analizar las historias de usuario, definir los flujos de navegación y diseñar los primeros wireframes y el prototipo visual.
Este sprint se enfocó en representar de manera clara la estructura de la interfaz y establecer la identidad visual del proyecto.

## Actividades realizadas
nálisis de historias de usuario y flujos de navegación esperados.  
Planeación y elaboración de **bocetos (wireframes)** de la página web.  
Disenioo del **prototipo interactivo** en Figma. 
Justificacion del diseño

## Evidencias
Las evidencias correspondientes se encuentran en /evidencias/ donde se muestra los wireframes, el prototipo, la guia para el usuario y su justificacion.
